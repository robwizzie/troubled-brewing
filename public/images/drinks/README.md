# Drink photos — drop your files here

Photos for the homepage drink features. Save JPGs in **this folder**
(`public/images/drinks/`) with these names and they appear automatically —
until a file exists, each frame shows a tasteful drawn motif instead, so
nothing ever looks broken.

| File name | Where it shows |
|---|---|
| `featured.jpg` | **This Week's Trouble** — the gold oval frame on the green panel |
| `iced-chai-latte.jpg` | Signature sips card #1 |
| `banana-split-coffee.jpg` | Signature sips card #2 |
| `coffee-flight.jpg` | Signature sips card #3 |

The signature-sips names follow the drink's menu name, lowercased with
spaces/punctuation as dashes (`Banana Split Coffee` → `banana-split-coffee.jpg`).
If the drinks on the menu change, name the file after the new drink the same way.

Once the admin CMS is connected, photos uploaded there (Quick Blocks →
Featured drink, or Menu Manager → item image) take priority over these files —
this folder is the quick way to get real photos up before that.

Export **JPG**, roughly 800–1200px on the long edge, subject centered (the
featured photo is cropped to an oval).
